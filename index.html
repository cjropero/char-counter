Elcontadordecaracteres.com es una herramienta web que le permite contar los caracteres y las palabras que componen un texto. Para empezar a utilizarlo lo único que debe hacer es copiar y pegar su texto o simplemente digitarlo sobre la caja de texto ubicada en la parte de arriba, de esta forma los contadores se actualizaran arrojando el número de caracteres y el número de palabras empleadas en el texto ingresado.


A continuación compartimos algunos de los límites de caracteres más conocidos:

  • La cantidad máxima para un mensaje en Twitter es de 140 caracteres.
  • La cantidad máxima para un para un mensaje de texto enviado desde Skype es de 160 caracateres.
  • Los Anuncios de google utilizan una cantidad máxima de 25 caracteres para el título del anuncio, 70 caracteres para el cuerpo del anuncio y 35 para la url que se muestra en el anuncio.
  • La cantidad máxima para una URL sobre Internnet Explorer es de 2083 caracteres.
  • La cantidad máxima para una etiqueta de título HTML debería ser de 70 caracteres.
